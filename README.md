## Technologies

JavaScript, React Native, Redux, Viro Media, Cloud Firestore

## Installation

```js
// install Viro Media app to use their sandbox

// install required libraries
npm install

// run app for development
npm run

// copy ngrok from terminal and paste into
// Viro Media app input field
```
